Intelligent Storage Management System

It allows to manage materials stock. It consists of a database, a server and materials containers with dedicated hardware, which act as clients to the server to update quantity of stored elements. The calculation of the stored quantity is done by weight measurement. Note: The app is hosted in Heroku, wait a few moments for the server to awake when trying the app.

Iot Control Panel

Dashboard that allows control of machines or devices connected to smartplugs. A WebSocket server is created for each smartplug, which allows to turn on, shutdown, or monitor power consumption of the connected element. The name and the IP address of each devices is stored in a database. Note: The app is hosted in Heroku, wait a few moments for the server to awake when trying the app. To test the app a WebSocket especially created for this endevour can be used, so there is no need to set up an actual machine. The URL for this WebSocket can be found in the webpage of the app.

IoT IMU with Visualization in the Browser and in Unity

Visualization of the measurements of an inertial measurement unit on a web client, served through a WebSocket server which receives measurements through serial port and exposes them. Imu rotation can be seen in the browser throguh graphics developed with three.js Additionally, an interface developed in Unity allows to visualize the rotation in a model. To try the app it is possible to connect to WebSocket server, which has stored values taken from an actual IMU and serves them upon request. The address of this WebSocket server can be seen in the webpage of the app.

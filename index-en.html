Intelligent Storage Management System

It allows to manage materials stock, automatically updating the quantity of the stored elements. It consists of a database, a server and materials containers. Each container has dedicated hardware, which calculates the ammount of items based on their weight. Other features are a web interface, users management, and a REST API.

Note: The app is hosted in Heroku, wait a few moments for the server to awake when trying the app.

Iot Control Panel

It is a dashboard that allows control of machines or devices connected to smartplugs. To do so, a WebSocket server is created for each smartplug, which allows to turn on, shutdown, or monitor power consumption of the connected element. The name and the address of each devices is stored in a database.

Note: The app is hosted in Heroku, wait a few moments for the server to awake when trying the app. To test the app a WebSocket especially created for this endevour can be used, so there is no need to set up hardware. The URL for this WebSocket can be found in the webpage of the app.

IoT IMU with Visualization in the Browser and in Unity

The goal of this app is to enable visualization of the measurements of an inertial measurement unit on a web client, served through a WebSocket server, which receives measurements through serial port and exposes them. Imu rotation can be seen in the browser throguh graphics developed with three.js

Additionally, an interface developed in Unity allows to visualize the rotation in a model. To try the app without hardware it is possible to connect to WebSocket server, which has stored values taken from an actual IMU and serves them upon request. The address of this WebSocket server can be seen in the webpage of the app.

Rocket Lander

Vertical rocket landing simulator modelled after SpaceX's Falcon 9 first stage rocket. The simulation was developed in Python 3.5 and written using OpenAI's gym environment. Box2D was the physics engine of choice and the environment is similar to the Lunar Lander.

This project is a fork, my contributions to it are: Fuzzy PID control class, and comparison metrics added to the main simulation file (fuel, time, goal). Additionally, data saving for later analisis.

Two Factor Authentication

Two factor authentication done by numerical password, typed with buttons and displayed on screen, and facial recognition.

The authentication success is simulated by turning a servomotor. Wrong entries of either input sound an alarm.

Autonomous Vehicle

An autonomous vehicle that navigates a track and overcomes a series of obstacles, including ramps, T intersections, U-turns, and parking. It is programmed using ROS and Python.

As platform, a kit with a Raspberry Pi was used. The vehicle has a camera (which is used for most of the navigation) and an ultrasonic sensor. The actuators of the robot are the motors in each wheel.

Image Processing for Rivet Analyzer

The goal of this project is to improve aircraft maintenance, by detecting damaged rivets in their fuselaje. A robotic platform is used to collect video of the rivets in the fuselaje of an airplane, then this footage is processed to extract suitable input for a neural network, which detects the rivets that need maintenance.

Sistema de Gestión de Almacenamiento Inteligente

Permite gestionar stock de materiales, actualizando automáticamente la cantidad de elementos almacenados. Consta de una base de datos, un servidor y contenedores de materiales. Cada contenedor posee hardware dedicado, el cual calcula la cantidad almacenada de un elemento en base al peso. Otras características del software son: interfaz web, administración de usuarios y una REST API.

Nota: La app está almacenada en Heroku, esperar unos momentos a que el servidor despierte al intentar acceder.

Panel de Control IoT

Se trata de un tablero que permite controlar máquinas o dispositivos conectados a enchufes inteligentes. Para ello, un servidor WebSocket es creado por cada enchufe que permite encender, apagar y controlar el consumo de energía del elemento conectado. El nombre y la dirección de cada dispositvo se almacena en una base de datos.

Nota: La app está almacenada en Heroku, esperar unos momentos a que el servidor despierte al intentar acceder. Para probar la app sin hardware se utiliza un WebSocket creado específicamente para estas pruebas.

IoT IMU con Visualización Web y en Unity

Esta aplicación tiene como objetivo la visualización de las mediciones de una unidad de medición inercial en un cliente web, servidas a través de un servidor WebSocket que toma las mediciones a través de un puerto serial. Puede verse la rotación de la IMU en gráficos desarrollados con three.js.

Adicionalmente, una interfaz desarrollada en Unity permite visualizar los valores de rotación en un modelo. Para probar la app sin hardware, es posible conectarse a un servidor WebSocket que creado específicamente para ello. Solo ingresar la dirección que se indica en la página web de esta app.

Aterrizador de Cohetes

Simulador de aterrizaje de cohetes verticales modelado a partir del cohete Falcon 9 de la primera etapa de SpaceX. La simulación se desarrolló en Python 3.5 y se escribió utilizando el entorno de gym de OpenAI. Box2D fue el motor de física elegido y el entorno es similar al del Lunar Lander.

Este proyecto es un fork de un proyecto existente, mis contribuciones son: clase de control PID de lógica difusa y métricas de comparación al archivo principal de simulación (tiempo, combustible, objetivo). Además, almacenamiento de estos datos para posterior análisis.

Autenticación de Dos Factores

Autenticación de dos factores realizada mediante contraseña numérica, introducida con botones y mostrada en pantalla, y reconocimiento facial.

El éxito de la autenticación se simula girando un servomotor. Las entradas incorrectas de cualquiera de los factores hacen sonar una alarma.

Vehículo Autónomo

Un vehículo autónomo que navega por una pista y supera una serie de obstáculos, incluyendo rampas, intersecciones en T, vueltas en U y estacionamiento. Está programado usando ROS y Python.

Como plataforma se utilizó un kit basado en Raspberry Pi. El vehículo consta de una cámara (la cual es usada para la mayor parte de la navegación) y un sensor ultrasónico. Motores en cada una de las cuatro ruedas son los acutadores del robot.

Procesamiento de Imágenes para Analizador de Remaches

El objetivo de este proyecto es mejorar el mantenimiento de una aeronave, mediante la detección de remaches defectuosos. Una plataforma robótica toma video de los remaches en el fuselaje del avión, luego un programa procesa estas imágenes y las convierte en entrada adecuada para una red neuronal, la cual detecta los remaches que necesitan mantenimiento.
